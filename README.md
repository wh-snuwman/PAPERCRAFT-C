
# PAPERCRAFT

페이퍼 크래프트는 온라인 서바이벌 게임입니다.
플레이어는 전투 혹은 협동 등 여러 선택을 할 수 있으며, 궁극적인 목표는
다른 플레이어와 여러 위혐요소로부터 가장 오래 생존하거나 가장 강력해지는 것입니다.

< 특징 > 
아기자기한 그래픽 / 이와 대조되는 난이도 있는 콘텐츠
다른 플레이어간 협력과 동맹가능 / 배신과 약탈도 가능
다양한 아이템과 스킬 / 군대, 무기, 폭격기
전투시 여러아이템과 팀원의 서포트 / 죽거나 전투중 이탈시 모든물품 소실
중독성 있는 게임플레이 / 강한 경쟁심

# WHAT
